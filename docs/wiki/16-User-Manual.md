# 16. Manual do Usuário

Guia rápido de operação para Gestores e Colaboradores.

## 1. Acessando o Sistema
- **Web**: Acesse `https://app.sisdavus.com.br` (URL exemplo).
- **Desktop**: Clique no ícone "SisDavus" na área de trabalho.
- **Login**: Insira seu email corporativo e senha.

## 2. Gestão de Patrimônio (Ativos)

### Cadastrar Novo Ativo 
1.  Vá para o menu **Patrimônio**.
2.  Clique em **Novo Ativo**.
3.  Preencha os dados obrigatórios:
    - **Nome**: Descrição curta (ex: Notebook Dell Latitude).
    - **Código**: Número da etiqueta de patrimônio (ex: PAT-00123).
    - **Status**: Selecione "Disponível".
4.  Clique em **Salvar**.

### Realizar Empréstimo (Checkout)
1.  Localize o ativo na lista.
2.  Clique no botão de **Ações (...)** e selecione **Checkout**.
3.  Selecione o **Colaborador** na lista.
4.  Defina a **Data Prevista** de retorno.
5.  Confirme. O status mudará para "Em Uso".

## 3. Controle de Estoque (Produtos)

### Registrar Entrada
1.  Vá para o menu **Estoque**.
2.  Clique no botão **+ Entrada**.
3.  Selecione o Produto (ex: Mouse USB).
4.  Insira a Quantidade (ex: 50).
5.  Clique em **Confirmar**. O saldo será atualizado automaticamente.

## 4. Dicas de Uso Offline
- O sistema salva tudo automaticamente mesmo sem internet.
- Um ícone de **"Nuvem Cortada"** aparecerá no topo quando estiver offline.
- Ao reconectar, aguarde o aviso verde **"Sincronização Concluída"** antes de fechar o app.
